# Ứng dụng Quản lý Tài nguyên OpenStack qua API - Lab 02 (NT533.Q21)

## Hướng dẫn cài đặt & Sử dụng

### Bước 1: Yêu cầu môi trường
- Python 3.8 trở lên.

### Bước 2: Cài đặt thư viện
Mở Terminal/Command Prompt và chạy lệnh sau để cài đặt giao diện `customtkinter` và thư viện gọi API `requests`:
```bash
pip install customtkinter requests
```

### Bước 3: Chạy ứng dụng
```
python main.py
```
